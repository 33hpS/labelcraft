# ⌨️ Горячие клавиши - Быстрый справочник v2.1

> 📌 Распечатайте или сохраните этот справочник рядом с рабочим местом оператора!

## 🎯 Основные операции

| Клавиша          | Функция        | Когда использовать     |
| ---------------- | -------------- | ---------------------- |
| **Ctrl+Z**       | ↶ Отменить     | Ошибка, нужно откатить |
| **Ctrl+Y**       | ↷ Повторить    | Вернуть отменённое     |
| **Ctrl+Shift+Z** | ↷ Повторить    | Альтернатива Ctrl+Y    |
| **Ctrl+A**       | ◻ Выбрать все  | Нужны все элементы     |
| **Ctrl+D**       | 📋 Дублировать | Скопировать выбранные  |
| **Delete**       | 🗑 Удалить      | Убрать лишние элементы |

## 🎯 Топ-5 горячих клавиш для оператора

### Для экономии времени:

1. **Ctrl+D** — Дублировать элемент (самая полезная!)

   - Копирует выбранные с автосмещением на X+15, Y+15
   - Экономит 3 клика за каждое дублирование

2. **Ctrl+Z** — Отменить ошибку

   - Отменяет последнюю операцию
   - Можно отменить неограниченное количество раз

3. **Ctrl+A** → затем кнопка **Выравнить**

   - Выбрать все элементы и выровнять их за раз
   - Идеально для создания симметричного макета

4. **G** — Включить сетку

   - Помощник при точном позиционировании
   - Работает на основе 20px сетки

5. **Ctrl+0** — Вернуться на 100%
   - Быстро вернуться к нормальному виду
   - Если случайно зумнули на 150% или 50%

## 🧠 Советы и трюки

### ✅ Быстрое создание 10 одинаковых элементов:

```
1. Создайте 1 элемент
2. Выберите его
3. Нажмите Ctrl+D 9 раз
4. Готово! 10 элементов за 10 нажатий
```

### ✅ Выравнивание 5 элементов:

```
1. Нажмите Ctrl+A (выбрать все)
2. Нажмите "Выравнить → По левому краю"
3. Все элементы выровняются за 1 клик
```

### ✅ Откат больших ошибок:

```
1. Нажимайте Ctrl+Z много раз
2. История не ограничена
3. Вернетесь к нужному состоянию
```

### ✅ Точное позиционирование:

```
1. Включите Сетку (G)
2. Увеличьте Zoom на 150-200%
3. Перетягивайте элементы вдоль сетки
4. Выключите Сетку (G) когда готово
```
