## 🎯 За 5 минут до первой работы

### Шаг 1: Откройте редактор (30 сек)

```
1. Перейдите на https://productlabelerpro-worker.sherhan1988hp.workers.dev
2. Нажмите на "Шаблоны" в меню
3. Нажмите "+ Новый шаблон"
→ Редактор открыт! ✅
```

### Шаг 2: Добавьте элементы (1 мин)

```
1. На холсте справа найдите кнопку "Добавить текст"
2. Нажмите → появился текстовый элемент
3. Дважды нажмите на него → отредактируйте текст
4. Нажмите "Добавить текст" еще 2 раза
→ На холсте 3 текстовых элемента ✅
```

### Шаг 3: Выровняйте (1 мин)

```
1. Нажмите Ctrl+A → все 3 элемента выделены
2. На панели инструментов нажмите кнопку "■" (По центру)
3. Все элементы выровнялись горизонтально ✅
```

### Шаг 4: Используйте undo (1 мин)

```
1. Перетягивайте элементы где угодно
2. Если ошибка → нажмите Ctrl+Z
3. Элементы вернулись! ✅
```

### Шаг 5: Сохраните (1 мин)

```
1. Нажмите кнопку "Сохранить" внизу
2. Введите название "Мой первый шаблон"
3. Выберите категорию
4. Нажмите "Создать"
→ Шаблон сохранен! ✅
```

## 🆕 Что нового в v2.1?

### ✨ 4 новые функции:

#### 1. **История буфера обмена** �

```
Ctrl+Shift+V → Выбери элемент из последних 5 копирований
```

- Копи элемент: `Ctrl+C`
- Позже открой историю: `Ctrl+Shift+V`
- Выбери нужный → вставится на холст

#### 2. **Магнитная сетка** 🧲

```
G → Включить/выключить
```

- Элементы прилипают к сетке при перемещении
- Точное позиционирование
- Кнопка на панели инструментов

#### 3. **Выбор размера сетки** 📏

```
Кликни на кнопку размера сетки → Выбери 10px, 20px или 50px
```

- 10px - Точное
- 20px - Среднее (по умолчанию)
- 50px - Быстрое

#### 4. **Оптимизированная история** 🗜️

- Максимум 500 шагов (раньше не было лимита)
- Автоматическое удаление дубликатов
- Меньше нагрузка на память браузера

## 💡 10 трюков для экономии времени

### 1️⃣ Дублируйте быстро

```
Нужно 5 одинаковых элементов?
1. Создайте 1 элемент
2. Выберите его
3. Нажмите Ctrl+D четыре раза
→ 5 элементов готовы за 5 секунд!
```

### 2️⃣ Выравнивайте за клик

```
Нужно выровнять много элементов?
1. Нажмите Ctrl+A (выбрать все)
2. Нажмите кнопку "По центру"
→ Все элементы центрированы!
```

### 3️⃣ Отмените всё

```
Испортили много элементов?
1. Нажимайте Ctrl+Z много раз
2. История не ограничена!
→ Вернетесь к идеальному состоянию
```

### 4️⃣ Заблокируйте логотип

```
Боитесь случайно сместить логотип?
1. Выберите логотип
2. Нажмите L
3. На логотипе появится замок 🔒
→ Теперь его невозможно переместить
```

### 5️⃣ Используйте сетку

```
Нужно идеально выровнять элементы?
1. Нажмите G → сетка вкл
2. Перемещайте элементы вдоль сетки
3. Нажмите G → сетка выкл
→ Элементы выровнены идеально!
```

### 6️⃣ Масштабируйте для деталей

```
Нужно поправить мелкие детали?
1. Нажмите + чтобы увеличить (до 200%)
2. Отредактируйте элементы
3. Нажмите Ctrl+0 чтобы вернуть 100%
→ Точное позиционирование готово!
```

### 7️⃣ Полный экран для проверки

```
Хотите увидеть макет в полный размер?
1. Нажмите F11 → полный экран
2. Проверьте всё
3. Нажмите Esc → вернулись в редактор
→ Ничего не отвлекает!
```

### 8️⃣ Копируйте элементы правильно

```
Хотите скопировать элемент между шаблонами?
1. В первом шаблоне: выберите элемент
2. Нажмите Ctrl+C
3. Откройте второй шаблон
4. Нажмите Ctrl+V
→ Элемент добавлен во второй шаблон!
```

### 9️⃣ Выбирайте несколько элементов

```
Хотите работать с несколькими элементами?
Способ 1:
- Нажмите Ctrl+A → выбрать все

Способ 2:
- Щелкните на первый
- Нажмите Ctrl+щелчок на остальные
→ Все выбраны, все операции сразу для всех!
```

### 🔟 История как защита

```
Хотите попробовать что-то рискованное?
1. Запомните, что сейчас показано в истории (например, "7/10")
2. Делайте всё что хотите
3. Нажимайте Ctrl+Z чтобы вернуться
→ Полная свобода творчества!
```

## 🐛 Если что-то не работает

### Проблема: "Не выбирается элемент"

```
Решение:
1. Нажмите на элемент еще раз
2. Если он маленький → увеличьте (нажмите +)
3. Если он заблокирован → нажмите L
```

### Проблема: "Элемент исчез"

```
Решение:
1. Нажмите Ctrl+Z несколько раз
2. Элемент вернулся!
3. История нескончаема - не бойтесь
```

### Проблема: "Слишком много элементов"

```
Решение:
1. Нажмите Ctrl+A → выбрать все
2. Нажмите Delete → удалить все
3. Или нажмите Ctrl+Z → отменить удаление
```

### Проблема: "Не видно всё на холсте"

```
Решение:
1. Нажмите Ctrl+0 → вернуть 100% масштаб
2. Или нажмите - несколько раз чтобы уменьшить
```

### Проблема: "Холст в беспорядке"

```
Решение:
1. Нажмите Ctrl+A → выбрать все
2. Нажмите "По левому краю"
3. Нажмите "Распределить"
→ Всё аккуратно выровнено!
```

